# gr0.2_MVP
Автономный Telegram-бот с устойчивой личностью, способный к естественному диалогу, самообучению и проявлению инициативы. 
Проект реализует модульную архитектуру искусственного интеллекта с памятью, эмоциональным интеллектом и системой целеполагания.

## Функционал:
- Устойчивая личность — бот «Гриша» с характером, историей и самосознанием
- Многоуровневая краткосрочная память (20+ сообщений)
- RAG-система — объединённая база знаний из датасета и выученных паттернов
- Система самообучения — автономное сохранение успешных паттернов и их повторное использование


## Установка:

1) Скопируйте репозиторий
2) Добавьте собственный датасет
3) В файле config.py напишите свой системный промпт
4) В файле main.py вставте свой токен из BotFather
5) Запустите файл main.py
